Game Concepts

Game Ideas From My Imagination

I may be in Class 10, but my mind is already full of game concepts. These are idea‑level "projects" — the kind of games I’d love to build one day as I learn more coding and game design.

01 • Neon Runner City

Endless runner with story‑based missions.

You play as a courier in a neon futuristic city, dodging traffic, jumping rooftops, and unlocking secret routes. The game combines speed with short story missions and choices.

  • Core idea: Smooth parkour + branching missions
  • Mechanics: Slide, wall‑run, slow‑motion jumps
  • Inspiration: Runner games + cyberpunk themes

02 • Sky Chess Arena

Strategy game mixing chess and arena combat.

Classic chess pieces come to life in a floating arena. Each piece has special abilities and cooldowns. You think like a chess player but fight like an arena gamer.

  • Core idea: Strategy + real‑time action
  • Mechanics: Ability combos & special moves
  • Inspiration: Chess, MOBAs, hero shooters

03 • Puzzle of Time

Story‑driven puzzle game about time loops.

You wake up in the same day again and again. Each loop lets you keep knowledge but not items. The challenge is to solve puzzles using information you learn over multiple loops.

  • Core idea: Learning‑based progression
  • Mechanics: Timelines, clues, hidden routes
  • Inspiration: Story games & puzzle adventures
